Title: AMAZON CLONE
Description:
       A fully responsive front-end clone of the Amazon homepage built to practice HTML and CSS layout techniques.  
       Features Used:
           Navigation Bar: It includes logo,search bar,account links,cart.
           Hero Section: Features the large background image with the "Find gifts for dads" message.
           Product Grid: Organized categories like Health & Personal Care and Electronics, Accessories, Clothing, Pet care.
           Footer: To know more about amazon.
        Technologies Used:
           HTML5: For the structure of the site.
           CSS3: For styling, darktheme, hovering of boxes.
           Font Awesome: For using icons like magnifing glass, cart, location.
        Learning Objective:
           "display:flex" to align all the boxes in the same row.
           "cursor:pointer" to change the mouse cursor into a hand icon with a pointing finger.
           "text-decoration:underline" property is used to draw a horizontal line directly beneath the text of an element.
            Implementing Inline CSS for unique background images in product boxes.
            
            Using frontend languages like html and css to build a webpage which is similiar to amazon webpage, in order to understand how these
            frontend languages help in creating a perfect webpage and to know about its properties.
